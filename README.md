# 🚀 TypeScript Assignment

এই রিপোজিটরিটি TypeScript-এর মৌলিক থেকে অ্যাডভান্সড লেভেলের বিভিন্ন টাস্ক ও তাদের সল্যুশন নিয়ে তৈরি।

এখানে মূল ফোকাস করা হয়েছে **Type Safety, Generics, Utility Types, এবং Conditional Logic** এর উপর।

---

## 📌 Overview

এই অ্যাসাইনমেন্টে মোট **১০টি স্ট্রাকচার্ড TypeScript টাস্ক** ইমপ্লিমেন্ট করা হয়েছে।

প্রতিটি টাস্কের লক্ষ্য ছিল:
- টাইপ সেফটি আরও শক্তিশালী করা  
- কোড পুনঃব্যবহারযোগ্য (Reusable) করা  
- ক্লিন ও মেইনটেইনেবল কোড লেখা  

বাস্তব প্রজেক্টে TypeScript কীভাবে কোডকে আরও নিরাপদ ও শক্তিশালী করে, তা এই প্রজেক্টে প্র্যাকটিক্যালভাবে দেখানো হয়েছে।

---

## 🧠 Key Concepts Covered

| Concept | Description |
|--------|-------------|
| Object Types & Optional Properties | Flexible data modeling using optional fields |
| Type Intersection (&) | Combining multiple types into one |
| Optional Chaining & Null Safety | Safe data access using `?.` and `??` |
| Unknown Type & Type Assertion | Handling uncertain data safely |
| Generics | Reusable functions with strict type safety |
| Generic Constraints | Restricting generics using `extends` |
| keyof Operator | Ensuring valid object key access |
| Literal Types & `as const` | Locking specific values |
| Mapped Types | Creating new types from existing ones |
| Conditional Types | Dynamic type selection based on conditions |
| Utility Types (Omit) | Creating safe public types by removing sensitive fields |
